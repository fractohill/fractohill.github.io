Hexo+GitHub搭建個人博客小記

碎山
2024-12-03
GitHub, Hexo, macOS

前言：由於博客搭建完成已經過了有一段時間了，很多操作記得不太清晰，只能零碎的紀錄下來，沒什麼參考意義。

一、安裝必要程序：Node.js, Git, Xcode, Hexo

1.Node.js

打開Node.js下載及安裝。（版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本）

2.Git

下載安裝程序

3.Xcode

在歷史版本)中找到自身Mac支持的最高版本，我這裡版本是12.3.1則對應的是Xcode 13.4.1，下載安裝即可。

4.hexo

使用npm安裝hexo
npm install -g hexo-cli

如果在尝试安装 Hexo 的过程中出现 EACCES 权限错误，请遵循 由 npmjs 发布的指导 修复该问题。

我在安裝時也出現了這個錯誤，當時嘗試在終端執行：
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
後執行：command -v nvm 查詢，輸出為nvm（node版本管理工具）就是成功安裝了。
nvm ls-remote 列出可用版本
nvm install v22.11.0 選擇安裝
nvm use v22.11.0 使用安裝好的版本

npm install -g hexo-cli 再次嘗試安裝hexo

二、準備就緒，開始搭建

  • 打開finder，新建一個博客文件夾
  • 打開終端，鍵入cd <博客文件夾位置> 或鍵入cd 並拖入文件夾，後按回車
  • 鍵入hexo init

三、發佈到GitHub

1.登入GitHub並新建一個與倉庫，設定為公開

倉庫名命名規則：username.github.io

2.配置SSH key

  • 終端鍵入以下：
    it config —global username ‘username’
    git config —global user.email ‘your email address’
    ssh-keygen -t rsa -C ‘your email address’
    按三次回車鍵就可生成ssh key，或輸入以下指令查看
    cat ~/.ssh/id_rsa.pub
  • 打開GitHub主頁 - 設置 - SSH and GPG keys - New SSH Key，把上一步的內容複製進去，名稱寫hexo（隨意）然後保存即可。
  • ssh -T git@github.com 鍵入測試，顯示You’ve successfully authenticated,but GitHub does not provide shell access. 就是添加成功了

3.修改配置文件

在本地的hexo文件夾裡面，找到_config.yml文件，打開修改deploy: 部分，修改並保存。

4.生成並部署網站

  • 生成hexo g
  • 部署hexo d
  • 提示輸入用戶名密碼時，密碼用token代替，因為密碼已經停止支持了

    在github - settings - developer settings - personal access tokens，添加一個token，時間設置為永久

  • 執行hexo g -d
  • 查看博客，打開username.github.io就可以看到部署好的網頁了

5.綁定自己的網域

  • settings - pages - custom domain，填入自有網域後保存
  • 打開官方文檔，在cloudflare上添加4個A紀錄
  • 添加www的cname紀錄，填帳號名字username.github.io，並開啟小雲朵☁️
  • 添加兩條TXT紀錄，驗證網域

  • 在本地hexo source 裡新增CNAME文件，寫上自己的網域，防止部署失效

四、寫在最後

至此一個博客的搭建就已經完成了，接下來的便是撰寫文章與發布，其他的部分諸如更換主題和基礎信息修改之類的日後有需要再寫吧。
