1. Hexo相關/目前使用主題casus-white

1.1 使用markdown嵌入本地圖片

為了方便圖片管理，新建一篇文章時，同時新建與文章同名文件夾的方式，存放該文相關圖片。

  • 打開_config.yml文件，找到post_asset_folder: false ，將false 設為true
  • 打開Terminal，輸入npm install hexo-renderer-marked --save
  • 打開並修改_config.yml文件，啟用後，會自動解析圖片路徑

1.2 TypeError: Cannot read properties of null (reading ‘utcOffset’)

生成靜態文件時出現此報錯，是因為最初改_config.yml裡內容的時候timezone寫了其他地區，改回Asia/Shanghai就好。
